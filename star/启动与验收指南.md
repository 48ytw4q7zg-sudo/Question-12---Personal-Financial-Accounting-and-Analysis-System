# 个人财务记账与分析系统 · 启动与验收指南

---

## 一、环境要求

| 工具 | 版本 | 验证命令 |
|------|------|----------|
| JDK | 21+ | `java -version` |
| Maven | 3.9+ | `mvn -v` |
| Node.js | 24 LTS | `node -v` |
| pnpm | 10+ | `pnpm -v` |
| MySQL | 8.4 | `mysql -V` |

所有工具必须安装好才能启动系统。缺少任一工具，一键启动脚本会报错。

---

## 二、如何启动系统

### 方法一：一键启动（推荐）

双击 `star/一键启动.bat`，脚本会自动完成：

1. **启动 MySQL 服务** — 自动检测并启动 MySQL80 服务
2. **初始化数据库** — 首次运行时执行 `system/sql/01-init.sql` 创建 `finance_db`（7表+种子数据）
3. **启动后端** — 新窗口运行 `mvn spring-boot:run`（端口 8080）
4. **启动前端** — 新窗口运行 `pnpm dev`（端口 5173，首次自动 `pnpm install`）
5. **打开浏览器** — 自动跳转 `http://localhost:5173`

也可用 PowerShell 版：`star/一键启动.ps1`（右键 → 使用 PowerShell 运行）

> **电脑重启后一键启动**：只需双击 `star/一键启动.bat` 即可，脚本会自动启动 MySQL + 初始化数据库 + 启动后端 + 启动前端 + 打开浏览器，无需任何手动操作。

### 方法二：手动启动

```bash
# 第一步：确保 MySQL 运行
net start MySQL80

# 第二步：初始化数据库（仅首次）
mysql -uroot -proot < system/sql/01-init.sql

# 第三步：启动后端
cd system/backend
mvn spring-boot:run

# 第四步：启动前端
cd system/frontend
pnpm install   # 仅首次
pnpm dev

# 第五步：打开浏览器
# 访问 http://localhost:5173
```

## 三、验收操作指南

### P0 功能验收（必须通过 · 6项）

---

#### P0-1：登录 / JWT

**注册：**
1. 打开 http://localhost:5173 → 自动跳转登录页
2. 点击「注册」切换到注册表单
3. 输入用户名（3-20位字母/数字/下划线）、密码（6-20位）、确认密码
4. 点击「注册」按钮
5. 验证：注册成功后提示，自动切到登录 Tab 并回填用户名

**登录：**
1. 输入 `zhangsan` / `123456` → 点击「登录」
2. 验证：成功跳转到 DashboardPage，顶部显示用户名 `zhangsan`，左侧显示导航菜单
3. 输入错误密码 → 验证：提示错误，不跳转
4. 输入不存在用户 → 验证：提示错误，不跳转

**JWT 校验：**
1. 登录后，浏览器 DevTools → Application → Local Storage 查看 `token` 字段
2. 手动删除 `token` → 刷新页面 → 验证：自动跳转回登录页（路由守卫生效）
3. 登录成功后刷新页面 → 验证：仍然保持登录状态（Pinia store 从 JWT payload 恢复用户信息）

## 四、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| MySQL 无法启动 | 服务名可能不是 MySQL80 | 检查 `sc query state= all` 找到实际 MySQL 服务名 |
| 数据库初始化失败 | root 密码不是 `root` | 修改 `application.yml` 中的 `DB_PASSWORD` |
| 后端启动报端口占用 | 8080 端口被其他程序占用 | 关闭占用程序或修改 `application.yml` 端口 |
| 前端 pnpm install 失败 | Node 版本过低 | 升级 Node.js 到 24 LTS |
| 登录后页面空白 | 后端未启动完成 | 等待后端窗口显示 "Started Application" 再操作 |
| 修改密码后无法登录 | 忘记新密码 | 用 `mysql -uroot -proot finance_db` 查看用户表 |
| 一键启动脚本报错 | 编码问题 | 确保使用 GBK 编码打开 bat 文件 |
