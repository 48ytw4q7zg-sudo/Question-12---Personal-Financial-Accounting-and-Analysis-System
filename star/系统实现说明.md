> 每个功能的实现路径：后端 → 前端，具体到每个代码文件。

## 技术栈概览

| 层 | 技术栈 | 版本 |
|---|---|---|
| 后端 | SpringBoot + MyBatis-Plus + MySQL + JJWT + BCrypt | 3.5.14 / 3.5.15 / 8.4 / 0.13.0 / cost=12 |
| 前端 | Vue 3 + Pinia + Element Plus + Axios + ECharts + Vite | 3.5.34 / 3.0.4 / 2.13.7 / 1.15.2 / 5.6.0 / 8.0.0 |
| 构建 | Maven 3.9 + pnpm 10.33 | JDK 21 + Node 24 LTS |

## 全栈通用基础设施

以下文件为所有功能共享，不属于某个具体 PRD 功能点：

### 后端通用

| 文件 | 位置 | 作用 |
|------|------|------|
| `Application.java` | `system/backend/src/main/java/com/example/finance/` | SpringBoot 启动入口 |
| `Result.java` | `...common/` | 统一 API 返回包装（code/message/data），所有 Controller 使用 |
| `BusinessException.java` | `...common/` | 业务异常类，Service 层抛出，Controller 不捕获 |
| `ErrorCode.java` | `...common/` | 错误码枚举（1001=用户名重复, 1002=密码错误, 3004=账户不存在...） |
| `GlobalExceptionHandler.java` | `...common/` | 全局异常处理器，13+ 种异常类型统一转 `Result.error()` |
| `EntityValidator.java` | `...common/` | 跨 Service 共享校验（validateAccount/validateCategory/defaultAndFormatYearMonth） |
| `CorsConfig.java` | `...config/` | CORS 配置（显式 origin allowlist，生产环境禁止 localhost） |
| `DataSourceConfig.java` | `...config/` | 数据源配置（生产环境校验非默认密码） |
| `JwtConfig.java` | `...config/` | JWT 配置属性（secret/expire，生产环境校验≥32字节密钥） |
| `MybatisPlusConfig.java` | `...config/` | MyBatis-Plus 分页插件 + 自动填充 createTime/updateTime |
| `WebMvcConfig.java` | `...config/` | 注册 LoginInterceptor，白名单=/api/user/login,/api/user/register,/api/health |
| `LoginInterceptor.java` | `...interceptor/` | JWT 校验拦截器，解析 Bearer token，提取 userId/role |
| `JwtUtils.java` | `...util/` | JWT 生成/解析工具（HMAC-SHA256，payload含userId+username+role） |
| `LoginRateLimiter.java` | `...util/` | 登录限流（Guava RateLimiter，每用户名每秒2次，防暴力破解） |
| `application.yml` | `system/backend/src/main/resources/` | 端口8080、数据库、JWT、Jackson时间格式等配置 |
| `01-init.sql` | `system/sql/` | 数据库初始化脚本（finance_db + 7表 + 种子数据 + 测试数据） |

## P0 功能实现详解

### P0-1：登录 / JWT

**后端：**

| 文件 | 位置 | 职责 |
|------|------|------|
| `UserController.java` | `...controller/` | POST `/api/user/register` + POST `/api/user/login` + POST `/api/user/change-password`，入参用 @Valid DTO |
| `UserService.java` | `...service/` | 接口：register/login/changePassword |
| `UserServiceImpl.java` | `...service/impl/` | 实现：BCrypt(12)加密 + 登录限流(2次/秒) + 不区分用户不存在和密码错误(防枚举) + DuplicateKeyException并发兜底 |
| `UserMapper.java` | `...mapper/` | BaseMapper<User>，简单 CRUD |
| `User.java` | `...entity/` | 用户实体（@TableName("user"), @TableId AUTO, password @JsonIgnore） |
| `UserLoginRequest.java` | `...entity/dto/` | 注册/登录入参 DTO（@NotBlank @Size @Pattern） |
| `LoginResponse.java` | `...entity/dto/` | 登录返参 DTO（token + userId + username + role） |
| `ChangePasswordRequest.java` | `...entity/dto/` | 改密入参 DTO（oldPassword + newPassword + confirmPassword） |
| `UserRole.java` | `...common/enums/` | 角色枚举（NORMAL=0, ADMIN=1） |
| `LoginInterceptor.java` | `...interceptor/` | 校验每个 /api/** 请求的 Bearer token，白名单除外 |
| `JwtUtils.java` | `...util/` | generateToken / parseTokenPayload（HS256，payload=userId+username+role） |
| `JwtConfig.java` | `...config/` | JWT 密钥/过期时间配置（生产校验≥32字节） |
| `LoginRateLimiter.java` | `...util/` | Guava RateLimiter 限流 |
| `WebMvcConfig.java` | `...config/` | 注册拦截器，白名单=/api/user/login,/api/user/register,/api/health |

**前端：**

| 文件 | 位置 | 职责 |
|------|------|------|
| `LoginPage.vue` | `...views/` | 登录/注册双Tab表单 + el-form rules 校验 + 注册成功自动切Tab回填用户名 |
| `api/user.js` | `...api/` | login()/register()/changePassword() API 调用 |
| `stores/user.js` | `...stores/` | setUser(含token写入localStorage)/clearUser/isLoggedIn/JWT恢复 |
| `router/index.js` | `...router/` | beforeEach守卫：JWT过期预检 + requiresAuth + 已登录跳首页 |
| `api/request.js` | `...api/` | 请求拦截器注入token + 401自动跳login + 开放重定向防护 |

## 关键设计决策

1. **transfer 不计入支出统计**：SQL WHERE type!=2 或 type IN (1,2) 但排除 transfer_id 非空，确保 Dashboard 支出不含转账
2. **BCrypt cost=12**：OWASP 推荐≥10，本项目用12更安全；已有$2a$10$兼容，新注册自动升级$2a$12$
3. **JWT 单次解析**：LoginInterceptor 调 parseTokenPayload 一次提取 userId+role，不做二次解析
4. **LambdaQueryWrapper**：简单查询统一用，仅 FOR UPDATE 和 JOIN/聚合用 XML/@Select
5. **前端 token 集中管理**：setUser() 统一写入 localStorage token，clearUser() 统一清除
6. **ECharts 按需加载**：echarts-lazy.js 动态 import，打包从~1MB降至~350KB
7. **响应式三断点**：≥992px侧栏200px / 768-991px折叠64px / <768px抽屉
