## 一、系统启动指南

### 1.1 环境要求

| 组件 | 版本要求 | 验证命令 |
|------|---------|---------|
| JDK | 21+ | `java -version` |
| Maven | 3.9+ | `mvn -v` |
| Node.js | 24 LTS | `node -v` |
| pnpm | 10+ | `pnpm -v` |
| MySQL | 8.4 LTS | `mysql --version` |

### 1.2 一键启动（推荐）

**Windows 用户**:
```bash
# 双击运行 star 文件夹中的启动脚本
star\一键启动.bat
```

**PowerShell 用户**:
```powershell
# 在 PowerShell 中执行
.\star\一键启动.ps1
```

启动脚本会自动完成以下操作:
1. 检测并启动 MySQL 服务（兼容 MySQL80/MySQL84/MySQL 等不同服务名）
2. 检查数据库 `finance_db` 是否存在，不存在则自动初始化
3. 设置后端环境变量（DB_PASSWORD、JWT_SECRET 等）
4. 启动后端服务（SpringBoot，端口 8080）
5. 安装前端依赖（首次运行）并启动前端服务（Vite，端口 5173）
6. 打开浏览器访问 http://localhost:5173

### 1.3 手动启动（备用方案）

如果一键启动失败，可按以下步骤手动启动:

**步骤 1: 启动 MySQL**
```bash
# 检查 MySQL 服务状态
sc query MySQL84

# 如果未运行，启动服务（需管理员权限）
net start MySQL84
```

**步骤 2: 初始化数据库**
```bash
# 进入项目根目录
cd Question-12---Personal-Financial-Accounting-and-Analysis-System-main

# 执行数据库初始化脚本
mysql -uroot -proot < system\sql\01-init.sql
```

**步骤 3: 启动后端**
```bash
cd system\backend

# 设置环境变量（Windows CMD）
set DB_USERNAME=root
set DB_PASSWORD=root
set JWT_SECRET=dev-finance-jwt-secret-key-2026-qxw-2501060122-must-be-at-least-32-bytes
set JWT_EXPIRE=604800000
set SERVER_PORT=8080

# 启动 SpringBoot
mvn spring-boot:run
```

**步骤 4: 启动前端（新开一个终端窗口）**
```bash
cd system\frontend

# 安装依赖（首次运行）
pnpm install

# 启动 Vite 开发服务器
pnpm dev
```

**步骤 5: 访问系统**

打开浏览器访问: http://localhost:5173

- **关闭后端**: 关闭标题为"后端-SpringBoot-8080"的命令行窗口

## 二、验收流程概述

### 2.1 功能优先级分类

本系统按照 PRD（产品需求文档）将功能分为三个优先级:

- **P0（必做）**: 6 个核心功能，必须全部实现并通过验收
- **P1（应做）**: 7 个重要功能，应当实现并通过验收
- **P2（可选）**: 5 个增强功能，可根据实际情况选择性验收

### 2.2 验收建议顺序

1. **先验收 P0**: 确保核心功能完整可用
2. **再验收 P1**: 验证系统功能完整性
3. **最后验收 P2**: 展示系统增强特性
4. **验收管理员功能**: 验证双角色设计
5. **运行单元测试**: 验证代码质量（140 个测试用例）

### 2.3 验收环境检查

在开始验收前，请确认:

- [ ] MySQL 服务已启动
- [ ] 后端服务已启动（http://localhost:8080/api/v1/health 可访问）
- [ ] 前端服务已启动（http://localhost:5173 可访问）
- [ ] 可以使用测试账号登录系统
- [ ] 浏览器控制台无报错（F12 打开开发者工具查看）

**预期结果**:

## 七、常见问题排查

### 问题 1: 启动脚本报错"MySQL 服务未找到"

**解决方案**:
```bash
# 查看实际的 MySQL 服务名
sc query state= all | findstr mysql

# 修改 star\一键启动.bat 中的 MYSQL_SERVICE 变量
# 例如: set "MYSQL_SERVICE=MySQL84"
```

### 问题 2: 后端启动失败"DB_PASSWORD 环境变量未设置"

**解决方案**:
```bash
# 手动设置环境变量
set DB_USERNAME=root
set DB_PASSWORD=root
set JWT_SECRET=dev-finance-jwt-secret-key-2026-qxw-2501060122-must-be-at-least-32-bytes

# 然后启动后端
mvn spring-boot:run
```

### 问题 3: 前端启动失败"pnpm 命令未找到"

**解决方案**:
```bash
# 安装 pnpm
npm install -g pnpm

# 验证安装
pnpm -v
```

### 问题 4: 登录失败"用户名或密码错误"

**解决方案**:
```bash
# 重新初始化数据库
mysql -uroot -proot < system\sql\01-init.sql

# 使用测试账号登录
# zhangsan / 123456
```
